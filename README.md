# Commander Bot

Telegram bot for TAMM office/WFH reminders with webhook support.

## Features
- Daily office/WFH reminders
- Parking log tracking
- Auto-restart on failure
- Health monitoring

## Deployment

### Railway (Recommended)
1. Push to GitHub
2. Connect to Railway
3. Set environment variables
4. Deploy

## Environment Variables
- `BOT_TOKEN` - Telegram bot token
- `CHAT_ID` - Telegram chat ID
- `PORT` - Server port (default: 3000)
